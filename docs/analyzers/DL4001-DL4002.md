# DL4001 / DL4002 - Cognitive Complexity

Measures how difficult a method is to understand, based on the [SonarSource Cognitive Complexity specification (v1.7)](https://www.sonarsource.com/docs/CognitiveComplexity.pdf).

| Rule   | Category   | Default Severity | Default Threshold |
| ------ | ---------- | ---------------- | ----------------- |
| DL4001 | Complexity | Info             | 5                 |
| DL4002 | Complexity | Warning          | 12                |

When a method's complexity exceeds the **moderate** threshold, `DL4001` is reported.
When it exceeds the **elevated** threshold, `DL4002` is reported instead (not both).

## Rationale

Cyclomatic complexity measures mathematical execution paths; it is primarily an analogue for testability. Cognitive complexity, conversely, measures the structural burden placed upon the human mind.

Every time control flow is nested—an `if` within a `for` within a `while`, we force the reader to maintain a deeper mental stack. We complect the current operation with the state and context of all enclosing scopes. The SonarSource specification deliberately penalises this structural nesting, whilst explicitly ignoring syntactic shorthands (such as null-coalescing operators or early returns) that reduce visual noise without adding mental overhead.

In the context of AI-assisted coding, maintaining this architectural boundary is vital. AI tools can synthesise immense blocks of logic instantaneously. Left unbounded, they naturally gravitate towards generating monolithic, deeply nested methods that are syntactically correct but utterly opaque to a human reviewer. By enforcing a rigid cognitive complexity budget, this analyser compels both developers and AI assistants to decompose logic into discrete, composable, and straightforward functions. The result is a system that remains fundamentally simple to read, review, and reason about.

## What counts

### Structural increments (+1, plus nesting penalty)

These add `1 + current nesting level`:

- `if`, ternary operator (`? :`)
- `switch` (statement or expression, counted once for the entire switch)
- `for`, `foreach`
- `while`, `do while`
- `catch`

### Hybrid increments (+1, no nesting penalty, but increases nesting)

- `else if`
- `else`

### Fundamental increments (+1 only, no nesting penalty)

- `goto`
- Each sequence of like binary logical operators (`&&`, `||`)
- Direct recursion (counted once per method, regardless of call sites)
- Catch filter clause (`when`)

### Nesting level increases (no increment, only affects deeper structures)

- Lambdas, anonymous methods, local functions

### What does NOT count

- `break` and `continue` (C# has no labelled jumps, so these are never counted)
- `try` and `finally` blocks
- Null-coalescing operators (`?.`, `??`, `??=`)
- Early `return`
- Method declarations themselves

## Examples

### Simple method (complexity: 1)

```csharp
string GetWords(int number)
{
    switch (number) // +1
    {
        case 1: return "one";
        case 2: return "a couple";
        case 3: return "a few";
        default: return "lots";
    }
} // total: 1
```

### Nested control flow (complexity: 9)

```csharp
void Process(bool a, bool b)
{
    try
    {
        if (a) // +1
        {
            for (int i = 0; i < 10; i++) // +2 (nesting=1)
            {
                while (b) { } // +3 (nesting=2)
            }
        }
    }
    catch (Exception e) // +1
    {
        if (b) { } // +2 (nesting=1)
    }
} // total: 9
```

### Logical operator sequences (complexity: 4)

```csharp
void Check(bool a, bool b, bool c, bool d, bool e, bool f)
{
    if (a // +1 (if)
        && b && c // +1 (sequence of &&)
        || d || e // +1 (sequence of ||)
        && f) // +1 (new sequence of &&)
    {
    }
} // total: 4
```

### Recursion (complexity: 2)

```csharp
int Factorial(int n)
{
    if (n <= 1) // +1
        return 1;
    return n * Factorial(n - 1); // +1 (recursion)
} // total: 2
```

### Lambda nesting (complexity: 2)

```csharp
void Run(bool b)
{
    Task.Run(() => // nesting level becomes 1
    {
        if (b) // +2 (nesting=1)
            Console.WriteLine();
    });
} // total: 2
```

## Configuration

### EditorConfig

```ini
# Set custom thresholds
dotnet_diagnostic.DL4001.cognitive_complexity_moderate_threshold = 5
dotnet_diagnostic.DL4001.cognitive_complexity_elevated_threshold = 12

# Adjust severity
dotnet_diagnostic.DL4001.severity = suggestion
dotnet_diagnostic.DL4002.severity = warning
```

### Attributes

Override thresholds per method or class:

```csharp
using DemoriLabs.CodeAnalysis.Attributes;

// Override moderate threshold only (elevated stays at 12)
[CognitiveComplexityThreshold(moderateThreshold: 10)]
void ComplexButNecessary() { ... }

// Override both thresholds
[CognitiveComplexityThreshold(moderateThreshold: 10, elevatedThreshold: 20)]
void VeryComplexButNecessary() { ... }

// Suppress entirely
[SuppressCognitiveComplexity]
void LegacyMethod() { ... }

// Apply to all methods in a class
[SuppressCognitiveComplexity]
class LegacyController { ... }
```
