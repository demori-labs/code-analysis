# DL2005 - Unused method parameter

Detects method parameters that are never referenced in the method body and could be safely removed.

| Rule   | Category | Default Severity | Code Fix |
| ------ | -------- | ---------------- | -------- |
| DL2005 | Usage    | Info             | No       |

## Why

Unused parameters add noise to method signatures. They mislead callers into thinking the value matters, increase cognitive load when reading the code, and make refactoring harder. Removing them simplifies the API surface.

## When the diagnostic fires

- A method, constructor, or local function has a parameter that is never referenced in the body

## When the diagnostic does NOT fire

- The parameter is used anywhere in the method body (including nested lambdas and local functions)
- The method is `virtual`, `abstract`, `override`, or `extern`
- The method implements an interface (explicitly or implicitly)
- The method has a `[DllImport]` attribute
- The method matches an event handler signature (`object sender, EventArgs e`)
- The method is `static void Main` (entry point)
- The parameter name starts with `_` (discard convention)
- The parameter has `out` modifier
- The parameter is the `this` parameter of an extension method

## Examples

### Violation

```csharp
public void Process(int count, string label) // DL2005 on 'label'
{
    Console.WriteLine(count * 2);
}
```

```csharp
public class Service
{
    public Service(int capacity) // DL2005 on 'capacity'
    {
        Console.WriteLine("Initialised");
    }
}
```

### Fixed

```csharp
public void Process(int count)
{
    Console.WriteLine(count * 2);
}
```

### No diagnostic

```csharp
// All parameters used
public void Process(int count, string label)
{
    Console.WriteLine($"{label}: {count}");
}

// Interface implementation
public class Handler : IHandler
{
    public void Handle(int code) { }
}

// Override method
public class Derived : Base
{
    public override void Process(int value) { }
}

// Event handler signature
public void OnClick(object sender, EventArgs e) { }

// Discard parameter
public void Callback(int _) { }

// Parameter used in nested lambda
public void Deferred(int x)
{
    Action a = () => Console.WriteLine(x);
}
```

## Configuration

```ini
# Adjust severity
dotnet_diagnostic.DL2005.severity = warning

# Disable entirely
dotnet_diagnostic.DL2005.severity = none
```
