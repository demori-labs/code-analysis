# DL2001 / DL2002 - ReadOnly parameter enforcement

These two rules work together to enforce the `[ReadOnly]` attribute on parameters.

| Rule   | Category | Default Severity | Description                          |
| ------ | -------- | ---------------- | ------------------------------------ |
| DL2001 | Usage    | Error            | Parameter marked `[ReadOnly]` must not be reassigned |
| DL2002 | Usage    | Error            | Incompatible attribute on parameter  |

## DL2001 — Parameter must not be reassigned

Reports when a `[ReadOnly]` parameter is reassigned within the method body. This includes simple assignments, compound assignments, and increment/decrement operations.

### Violation

```csharp
using DemoriLabs.CodeAnalysis.Attributes;

public void Process([ReadOnly] int count)
{
    count = 10; // DL2001
    count += 5; // DL2001
    count++; // DL2001
}
```

### No diagnostic

```csharp
public void Process([ReadOnly] int count)
{
    var local = count + 1; // OK — reading the parameter
    Console.WriteLine(count); // OK — passing as argument
}
```

### Primary constructor parameters

The rule also applies to primary constructor parameters on classes and non-readonly structs:

```csharp
public class Widget([ReadOnly] int count)
{
    public void Reset()
    {
        count = 0; // DL2001
    }
}
```

### Code fix

**Introduce local variable for '{name}'** — creates a new local variable and redirects all subsequent references:

```csharp
// Before
public void Process([ReadOnly] int count)
{
    count += 5;
    Console.WriteLine(count);
}

// After
public void Process([ReadOnly] int count)
{
    var countLocal = count + 5;
    Console.WriteLine(countLocal);
}
```

Compound assignments are expanded: `x += 5` becomes `var xLocal = x + 5`. Increment/decrement operations are expanded: `x++` becomes `var xLocal = x + 1`.

## DL2002 — Incompatible attribute on parameter

Reports when `[ReadOnly]` or `[Mutable]` is applied to a parameter where it conflicts with the parameter's semantics or with each other.

### Incompatible cases

| Case | Why incompatible |
| --- | --- |
| `[ReadOnly]` on `ref` parameter | Exists specifically to allow mutation |
| `[ReadOnly]` on `out` parameter | Must be assigned by the callee |
| `[ReadOnly]` on `in` parameter | Already read-only by the language |
| `[ReadOnly]` on record class primary constructor | Records generate properties, not fields |
| `[ReadOnly]` on record struct primary constructor | Records generate properties, not fields |
| `[ReadOnly]` on readonly struct primary constructor | Parameters are already readonly |
| `[Mutable]` on record class primary constructor | Records generate properties, not fields |
| `[Mutable]` on record struct primary constructor | Records generate properties, not fields |
| `[Mutable]` on readonly struct primary constructor | Parameter is already readonly |
| `[ReadOnly]` + `[Mutable]` on same parameter | Contradictory — cannot be both |

### Violation

```csharp
using DemoriLabs.CodeAnalysis.Attributes;

// [ReadOnly] on ref/out/in
public void M([ReadOnly] ref int x) { }            // DL2002
public void N([ReadOnly] out int x) { x = 0; }     // DL2002
public void P([ReadOnly] in int x) { }             // DL2002

// [ReadOnly] on records and readonly structs
public record Person([ReadOnly] string Name);               // DL2002
public record struct Counter([ReadOnly] int Value);         // DL2002
public readonly record struct Point([ReadOnly] int X);      // DL2002
public readonly struct Size([ReadOnly] int Width);          // DL2002

// [Mutable] on records and readonly structs
public record Widget([Mutable] int Count);                  // DL2002
public record struct Pair([Mutable] int A);                 // DL2002
public readonly struct Vec([Mutable] int X);                // DL2002
public readonly record struct Coord([Mutable] int X);       // DL2002

// [ReadOnly] + [Mutable] conflict (order does not matter)
public void Process([ReadOnly][Mutable] int id) { }        // DL2002
public void Process([Mutable][ReadOnly] int id) { }        // DL2002
public class Service([ReadOnly][Mutable] int count);        // DL2002
```

### Allowed

```csharp
// [ReadOnly] on class/struct primary constructor — valid
public class Widget([ReadOnly] int count);          // No DL2002
public struct Point([ReadOnly] int x);              // No DL2002

// [Mutable] on class/struct primary constructor — suppresses DL2003
public class Widget([Mutable] int count);           // No DL2002
public struct Point([Mutable] int x);               // No DL2002

// [ReadOnly] on method parameters — valid
public void Process([ReadOnly] int id) { }          // No DL2002
```

### Code fix

**Remove conflicting attribute** — removes the incompatible attribute from the parameter.

```ini

dotnet_diagnostic.DL2002.severity = warning

# Disable entirely
dotnet_diagnostic.DL2001.severity = none
dotnet_diagnostic.DL2002.severity = none
```
